# Ayurvedic-Remedy-Recommender
Ayurvedic-Remedy-Recommender is an Android application that helps users find suitable Ayurvedic remedies based on their symptoms. It provides detailed information on formulations, supports offline access, and allows users to save favorites, making classical Ayurvedic knowledge easily accessible in a digital form.

Features:
-Implemented a symptom-based recommendation system using keyword and synonym matching.
-Enabled users to save, organize, and revisit preferred remedies.
-Integrated user authentication with login and sign-up functionality.
-Included an informational module to describe the application and its objectives.

##Tech Stack:
Platform: Android (Java)
UI Framework: XML layouts
Data Handling: Static datasets with mock search logic
Storage: Room Database (for favorites)
Navigation: Intent-based activity flow
